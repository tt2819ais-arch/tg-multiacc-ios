# TG MultiAcc — iOS

Многоаккаунтный помощник для Telegram. SwiftUI + [TDLibKit](https://github.com/Swiftgram/TDLibKit) (TDLib).

## Возможности
- Хранит много аккаунтов локально, сессии TDLib персистятся между запусками — повторно входить не нужно.
- Добавление аккаунта по номеру → коду → 2FA-паролю.
- **Массовая подписка** на канал/чат по `@username` или `t.me/...` ссылке от всех аккаунтов сразу.
- **Массовая отправка сообщения** пользователю/чату от всех аккаунтов сразу (с задержкой anti-flood).
- **Помощник по розыгрышам**: открывает указанный чат, показывает inline-кнопки последнего сообщения, по нажатию жмёт выбранную кнопку со всех аккаунтов одновременно.
- **Капчи / квизы**: режим «по очереди» — приложение проводит тебя через каждый аккаунт, ты вводишь ответ, переходим к следующему.
- Тёмная тема, жирные шрифты, минималистичный UI.

## Сборка через GitHub Actions

При каждом push на `main` запускается workflow `.github/workflows/ios.yml`, который:
1. Поднимает macOS-runner, устанавливает XcodeGen.
2. Генерирует `.xcodeproj`.
3. Резолвит SPM-зависимости (TDLibKit + TDLibFramework xcframework).
4. Собирает приложение под `iphoneos` без подписи.
5. Упаковывает `.app` в `Payload/` и архивирует в `TGMultiAcc.ipa`.
6. Загружает `.ipa` как artifact и (при тегах `v*`) — как GitHub Release.

`.ipa` будет на странице запуска workflow, во вкладке **Artifacts**.

## Установка `.ipa` на iPhone (без Apple Developer аккаунта)

`.ipa` **unsigned**. Варианты установки:

| Способ                                                        | Подпись       | Срок жизни | Нужен JB |
| ------------------------------------------------------------- | ------------- | ---------- | -------- |
| [TrollStore](https://github.com/opa334/TrollStore)            | не нужна      | бессрочно  | нет\*    |
| [AltStore](https://altstore.io) / SideStore                   | Apple ID      | 7 дней     | нет      |
| [Sideloadly](https://sideloadly.io)                           | Apple ID      | 7 дней     | нет      |

\* TrollStore работает только на iOS 14.0–16.6.1 (с нюансами по моделям).

Самый удобный для долговременного использования — TrollStore, если поддерживается версия iOS.

## Локальная разработка

```bash
brew install xcodegen
xcodegen generate
open TGMultiAcc.xcodeproj
```

При первом открытии Xcode скачает TDLibFramework (~300 МБ).

## Конфигурация

`api_id` / `api_hash` зашиты в [`Sources/Services/TDConfig.swift`](Sources/Services/TDConfig.swift). Для production-сборки замени их на свои с https://my.telegram.org.

## ⚠️ Дисклеймер

Массовая подписка и рассылка с многих аккаунтов нарушает [Telegram Terms of Service](https://telegram.org/tos) и может привести к **FloodWait** или **постоянному бану** аккаунтов. Используй на свой страх и риск, на одноразовых аккаунтах. Автор и репозиторий не несут ответственности.
